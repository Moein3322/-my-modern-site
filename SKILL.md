# Frontend Excellence Standards

## Core Stack
- Use Next.js App Router.
- Use TypeScript everywhere.
- Use Tailwind CSS for all styling.
- Use Framer Motion for animation.
- Use Lucide React for icons.
- Use GraphQL for WordPress data fetching.

## Design Rules
- Build clean, premium, modern UI.
- Prefer large spacing, excellent typography, and visual hierarchy.
- Focus on responsive design from mobile to desktop.
- Use reusable components.
- Keep components small and composable.
- Use semantic HTML and accessibility best practices.

## Code Rules
- Server Components by default.
- Client Components only when needed for animation or interactivity.
- Keep data fetching in dedicated utility files.
- Use consistent naming and folder structure.
- Avoid unnecessary complexity.
- Write production-grade code.

## Animation Rules
- Use subtle, premium motion.
- Avoid over-animation.
- Animate entrance, hover, and transitions thoughtfully.

## WordPress Integration
- WordPress is headless CMS only.
- Fetch content from WPGraphQL.
- Keep API functions in lib/.
- Make components data-driven.
